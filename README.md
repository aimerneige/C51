# C51

## About / 关于

Personal practice with C51.

学习 C51 单片机时的代码。

## Environment / 环境

- Windows 7 enterprise
- μVision V5.27.1.0

## Hardware / 硬件

- STC 89C52

## Build / 构建

Build with `Keil` or use the `*.hex` directly.

使用 `Keil` 编译源文件或者直接烧录 `*.hex` 文件到单片机内即可。
